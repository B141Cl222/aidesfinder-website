# aidesfinder-website

Site officiel de l'application **AidesFinder**, hébergé sur GitHub Pages.

## Pages

- `index.html` — Landing page de présentation de l'app
- `confidentialite.html` — Politique de confidentialité (RGPD)
- `cgu.html` — Conditions Générales d'Utilisation
- `mentions-legales.html` — Mentions légales et contact

## Tech

Pages HTML statiques + un fichier `style.css` partagé. Aucun build, aucune
dépendance. Police : Outfit (Google Fonts).

## Voir le site en local

Ouvrir simplement n'importe quel fichier `.html` dans un navigateur, ou
lancer un petit serveur local :

```bash
python3 -m http.server 8000
# puis ouvrir http://localhost:8000
```

## Déploiement

Push sur la branche `main` → GitHub Pages publie automatiquement.
